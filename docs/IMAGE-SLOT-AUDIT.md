# BM klus BV — Аудит image-слотов и семантический бриф по изображениям

**Дата:** 2026-03-13  
**Охват:** Все публичные маршруты, общие компоненты, источники данных  
**Статус:** Аудит только для чтения — без изменений в коде

---

## 1. Общая сводка

### Общее качество image-слотов
На сайте **~95 уникальных значимых image-слотов** на всех публичных страницах (не считая ~200 фотографий в проектных галереях, которые динамичны для каждого проекта). Все слоты используют реальные фотографии `.webp` — **ноль стоковых фото, ноль AI-сгенерированных изображений, ноль сломанных плейсхолдеров** в активном использовании. Alt-тексты последовательно заполнены и описательны. Формат изображений единообразно `.webp` (одно исключение `.jpg`).

### Основные сильные стороны
- Реальные фотографии проектов используются последовательно в hero-секциях и портфолио
- Хорошая дисциплина alt-текстов с контекстом локации и типа услуги
- Пары «до/после» доступны для всех 12 проектов
- Фотографии деталей/текстур существуют для материалов и отделок (sierpleister, ondergronden)

### Основные слабые стороны
1. **Повторное использование hero-изображений** — одна и та же фотография проекта (Dordrecht) используется для hero `/gevelisolatie/` И для всех страниц `/gevelisolatie/[location]/` (Rotterdam, Den Haag и т.д.)
2. **Три cluster-страницы без изображений** — `/gevelisolatie/afwerkingen/`, `/gevelisolatie/kosten/`, `/gevelisolatie/rc-waarde-dikte/` используют только CSS-градиенты в hero
3. **Изображения карточек услуг (`dienst-*.webp`)** используются в нескольких контекстах (страница diensten, секция услуг на главной, секция ETICS, табы afwerkingen) — визуальная монотонность
4. **`/buiten-stucwerk/` бедна изображениями** — всего 2 фотографии на целую money-страницу
5. **Секция услуг на главной** использует нативные теги `<img>` (не `next/image`), дублируется 2× в DOM (десктопный + мобильный рендер)
6. **Карточки слоёв ETICS** (главная страница) имеют однословные alt-тексты ("Stucwerk", "Steenstrips")

### Основные возможности
- Добавить hero-фотографии на 3 cluster-страницы без изображений
- Использовать фотографии проектов, специфичные для локации, на страницах `/gevelisolatie/[location]/`
- Добавить фотографии процесса/werkwijze на `/buiten-stucwerk/`
- Улучшить конкретность alt-текстов на карточках слоёв ETICS и карточках услуг

- Все 12 проектов имеют обе галереи «voor» и «na» — ✅ полно
- Галереи «до» (126 итого) значительно крупнее, чем галереи «после» (74 итого) — некоторые проекты имеют в 3 раза больше фото «до», чем «после»
